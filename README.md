# Tic-Tac-Toe Telegram Bot

## How to Play

In any chat with the bot, send:

```
/tictactoe
```

## Rules

- Two players take turns on a 3x3 grid
- The player who starts the game plays as ❌ and goes first
- The other player joins as ⭕
- Players tap the grid buttons to place their mark
- First to get 3 in a row (horizontal, vertical, or diagonal) wins
- If all 9 cells are filled with no winner, it's a draw

## Group Chat

1. Send `/tictactoe` to create a game
2. Another player taps "Join as ⭕"
3. Take turns tapping empty cells

## Private Chat

Send `/tictactoe` to play against the bot AI.
